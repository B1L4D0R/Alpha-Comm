# 🛰️ ALPHA COMM: Tactical Mesh Messenger

> **STATUS:** OPERATION ALPHA RELAY - ACTIVE
> **VERSION:** 1.1.0-ELITE
> **PROTOCOL:** ARP (Alpha Relay Protocol)

**Alpha Comm** is a resilient, decentralized communication system designed for scenarios where civil or military infrastructure is compromised. It leverages native smartphone hardware (Bluetooth, Wi-Fi Direct, and Ultrasonic Signaling) to build a peer-to-peer mesh network.

![Alpha Comm Feature Graphic](https://antigravity-artifacts.s3.amazonaws.com/3f64b6f6-d894-4e75-9dc8-9975105b990c/alpha_comm_feature_graphic_cinematic_1775608839341.png)

## ⚡ Key Features

*   **Alpha Relay Protocol (ARP):** Intelligent message chunking and multi-hop routing for extended range.
*   **Stealth Handshake:** Zero-config secure pairing using ultrasonic/NFC proximity.
*   **100% Offline Resilience:** Progressive Web App (PWA) architecture with full cache-first reliability.
*   **Tactical Radar:** Visual representation of mesh nodes, signal strength, and battery drift.
*   **Emergency SOS:** Global signaling with optional "Wipe" (Tactical Data Self-Destruction).

## 🛠️ Technology Stack

- **Framework:** [Vite](https://vitejs.dev/) + [React](https://reactjs.org/) + [TypeScript](https://www.typescriptlang.org/)
- **Styling:** [Tailwind CSS](https://tailwindcss.com/) (Obsidian Black Theme)
- **Animations:** [Framer Motion](https://www.framer.com/motion/)
- **PWA:** [Vite PWA Plugin](https://vite-pwa-org.netlify.app/)
- **Icons:** [Lucide React](https://lucide.dev/)

## 🚀 Deployment

The system is optimized for **Cloudflare Pages** and **standalone home-screen installation**.

```bash
# Clone the repository
git clone https://github.com/B1L4D0R/Alpha-Comm.git

# Install dependencies
npm install

# Run tactical simulation
npm run dev
```
